--- layout: page title: backup --- Small script to generate fast backups of files and directories. If the first argument is a directory it creates a gzip'd tar archive of it with a name like `filename_$(date +"%F_%H-%M-%S").tgz` . Otherwise it copies the file to `filename_$(date +"%F_%H-%M-%S")` . Download it directly from your terminal: {% highlight bash %} wget /wp-content/uploads/pipapo/scripts/backup {% endhighlight %}
