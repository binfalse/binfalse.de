--- layout: page title: check_imapds.sh --- This script checks the number of imap-deamons and warns if it reaches the maximum number. We had some problems with the number of imap-deamons, so I wrote this small script. It parses the Courier configuration file located in `/etc/courier/imapd-ssl` and warns if only 5 deamons are left and alerts if the maximum number is reached. Here is the code: [cc lang="bash" file="pipapo/monitoring/check_imapds.sh"][/cc] Please consider to take a look at my general setup notes.
