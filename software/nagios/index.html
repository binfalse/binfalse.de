--- layout: page title: 'Monitoring plugins' --- Here I publish some monitoring-plugins written by me, you can use them with Nagios or Icinga. Please consider to take a look at my setup notes.
check_catalyst_fans.pl
This script monitors the fan status of a Cisco catalyst via SNMP.
check_catalyst_flash.pl
This script monitors the flash status of a Cisco catalyst via SNMP.
check_catalyst_load.pl
This script monitors the load of a Cisco catalyst via SNMP.
check_catalyst_mem.pl
This script monitors the memory of a Cisco catalyst via SNMP.
check_catalyst_temp.pl
This script monitors the temperature of a Cisco catalyst via SNMP.
check_extreme_fans.pl
This script monitors the fans of an Extreme Networks device via SNMP.
check_extreme_fdb.pl
This script monitors the FDB of an Extreme Networks device via SNMP.
check_extreme_mem.pl
This script monitors the memory of an Extreme Networks device via SNMP.
check_extreme_powersupply.pl
This script monitors the power supply of an Extreme Networks device via SNMP.
check_extreme_temp.pl
This script monitors the temperature of an Extreme Networks device via SNMP.
check_hp_health.pl
Monitor the health state of you HP ProLiant.
check_imapds.sh
This script checks the number of imap-deamons and warns if it reaches the maximum number.
check_mount.pl
A Nagios-plugin to check if a device is mounted.
check_pykota.sh
This script checks whether PyKota is running.
check_UBC.pl
A Nagios-plugin to monitor fail-counts of OpenVZ containers.
check_url_shortener.pl
Check if an URL shortener does its job.
check_wp.php
Monitoring the WordPress up-to-date status.
