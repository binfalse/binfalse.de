For those as lazy as me: echo "deb http://apt.binfalse.de binfalse main" > /etc/apt/sources.list.d/binfalse.list gpg --recv-keys 08E66E72A83A9871CD493441E81BC3078D2DD9BD --keyserver pgp.mit.edu gpg --export 08E66E72A83A9871CD493441E81BC3078D2DD9BD | apt-key add - aptitude update All packages I provide are prefixed with `bf-`.
