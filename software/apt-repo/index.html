--- layout: page title: Apt-Repo --- I'm maintaining an apt repo (using [reprepro](https://mirrorer.alioth.debian.org/)). Just create a file `/etc/apt/sources.list.d/binfalse.list` containing the following: deb http://apt.binfalse.de binfalse main Packages are signed with my [GPG key 8D2DD9BD](https://binfalse.de/gpg-key/index.html). For those as lazy as me: echo "deb http://apt.binfalse.de binfalse main" > /etc/apt/sources.list.d/binfalse.list gpg --keyserver pool.sks-keyservers.net --recv-keys 08E66E72A83A9871CD493441E81BC3078D2DD9BD gpg --export 08E66E72A83A9871CD493441E81BC3078D2DD9BD | apt-key add - aptitude update The new approach for getting the gpg key seems to be: wget -O /etc/apt/trusted.gpg.d/binfalse.asc https://binfalse.de/gpg-key/index.html All packages I provide are prefixed with `bf-`.
